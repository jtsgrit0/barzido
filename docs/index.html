✨ 주요 기능

🗺️ 지도에서 한눈에 확인

구글 지도 위에서 내 주변의 모든 공연장과 펍을 실시간으로 확인하세요. 라이브 공연장은 네온핑크, 펍은 시안색 마커로 한눈에 구분할 수 있습니다.

📅 다가오는 공연 일정

홍대, 이태원의 핫한 라이브 공연 일정을 미리 확인하고 계획하세요. 다가오는 이벤트를 놓치지 마세요.

🏷️ 지역·카테고리 필터링

원하는 지역과 업종을 필터링해서 나에게 맞는 공간을 빠르게 찾아보세요. 검색 기능으로 원하는 장소를 즉시 찾을 수 있습니다.

🌙 다크 테마 UI

밤에 사용하기 편안한 네온톤 다크 테마로 눈이 피곤하지 않게 앱을 사용하실 수 있습니다. 밤의 분위기를 살린 디자인이 특징입니다.

☁️ Firebase 실시간 동기화

Firebase Firestore로 실시간 데이터가 항상 최신 상태로 유지됩니다. 새로운 공연 정보가 즉시 반영됩니다.

🚀 지속적인 업데이트

사용자 리뷰, 티켓 예약, 친구와 함께가기 기능 등 더 많은 기능이 계속 추가될 예정입니다. 전국 도시로의 확장도 계획중!
